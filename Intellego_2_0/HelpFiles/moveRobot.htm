Moving the Robot in the SimWorld

Once the Robot is properly initialized (meaning you can see the SimRCX Object in the SimWorld), it can be moved anywhere in the SimWorld with the mouse, similarly to Wall and Ground/Intersection Objects.

NOTE: The SimRCX Object portion of the robot is displayed in the image below:

To move the SimRCX Object:

1.) Place the cursor over top the SimRCX Object. (area within the rectangle)

2.) Press the LEFT mouse button DOWN, a selection ring will appear around the robot. (image below)

3.) Drag (combination of holding down the mouse button and moving the mouse) the SimRCX Object to a desired location on the SimWorld canvas

4.) Release the PRESSED LEFT mouse button when a suitable location is found for the object
