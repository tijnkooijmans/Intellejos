Opening LightSensor Prop Dialog

Light Properties can ONLY be set once a Controller File is opened in a Simulator window. As explained in 'Running a Simulation', part '8.)', when a Contoroller File is opened, a 'LightSensor Properties' dialog popup will appear, reminding you to not forget to map colours to numbers in the SimWorld.

NOTE: Every colour that has nothing in the 'Range' and 'Error' column of the LightSensor Properties dialog, if read by a LightSensor will return 911.

To subsequently open the LightSensor Properties dialog click: Options->LightSensor Prop.
