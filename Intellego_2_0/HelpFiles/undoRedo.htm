Using Undo/Redo

Introduction:

The main reason why UNDO and REDO commands were added is assist in retracting mistakes

EXAMPLE: The user positions a Wall Object absolutely and moves the object by accident, to avoid the long process of setting the absolute coordinates again the Undo command will retract that accidental move)

The following list covers what exactly UNDO/REDO cover:

--> addition of ALL SimWorld Objects

--> removal of ALL SimWorld Objects

--> position change of ALL Wall and Ground/Intersection Objects

--> colour change of ALL Ground/Intersection Objects

How to Use:

--> To use UNDO in an active SimWorld click: Options->Undo

--> To use REDO in an active SimWorld click: Options->Undo
