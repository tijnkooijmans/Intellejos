Running a Simulation

Start by opening a Code Editor window to complie your Controller File:

1.) To open click: Tools->Code Editor

Open, Save and Java Compile the Controller File which you want to run on the Simulator:

2.) To open click: File->Open

3.) To save click: File->Save

4.) To compile with Java click: Simulator->Complie

Next open a Simulator window:

5.) To open click: Tools->Simulator

NOTE: Only ONE Simulator window is allowed to opened at any given time

Select a SimWorld for the simulation run:

6.) To open SimWorld File: Controls->Open SimWorld

Open the Controller File in the Simulator:

7.) To add the Contoller File: Controls->Add Robot

8.) A 'Light Properties' dialog popup will appear, initialize it and click OK (click: learn about Light Properties)

9.) A menu will popup in the 'Property Settings' area, titled 'Initialize Robot,' fill in the desired values and click OK (click: learn how to Initialize a Robot).

Now everything is in order to start the simulation:

10.) To START the simulation: Controls->Start Simulation

11.) To STOP the simulation: Controls->Stop Simulation

NOTE: The 'Simulation Status' within the Simulation window will indicate whether a simulation is currently running.
