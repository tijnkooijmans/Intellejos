Creating a SimWorld File

Start by opening a Simulator window:

1.) To open click: Tools->Simulator

NOTE: Only ONE Simulator window is allowed to opened at any given time

All SimWorld Files will be based on previously created SimWorld Files. To begin creating a new instance of a previous SimWorld open an existing SimWorld.

2.) To open a SimWorld: Simulator->Controls->Open SimWorld (ie. ->BasicWorld)

NOTE: ALL of the SimWorld Files should be stored in the simworlds directory

There are two types of objects which can be created in the SimWorld:

1.) Walls

2.) Ground/Intersections

To learn how to create/modify objects follow the links below.

After you have populated your SimWorld with objects you may want to save your SimWorld:

3.) To save the SimWorld File: Controls->Save SimWorld Instance

A 'Save Instance of this SimWorld' dialog will popup, denoting the path where the SimWorld will be stored as well as an input field for the instance of the SimWorld.

NOTE: The file name of the save must be a different name opened SimWorld as well the name must differ from ALL files in the simworlds directory. Error prompts will indicate invalid file names.
