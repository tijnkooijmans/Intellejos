Light Properties

About:

To allow the LightSesors on the Robot to read colours properly, in the simulated environment of the SimWorld, colours must be mapped to a range of numbers. This is accomplished through a LightSensor Properties dialog. (image below)

NOTE: EVERY unmapped colour that is interpreted by the LightSensors will return a value of 911.

This editor contains a list of all posible colours that may be interpreted by the Robot's LightSensors. Also the listed 11 colours are the ONLY available colours used to fill SimWorld Objects.

NOTE: Only the Path and Ground/Intersection Objects and SimWorld Background may be read by the Robot's LightSensors.

The Buttons:

--> the functionality of the Open and Save buttons is described in 'Saving/Opening LightSensor Colour Values.' (link below)

--> the Clear button clears ALL of the colour check boxes
