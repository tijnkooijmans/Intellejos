Using the Lock Tool

Introduction:

The Lock Tool is located on the main tool bar in the Simulator window. (image below)

The main purpose of the Lock Tool is to prevent the user from making changes to the SimWorld. This is useful during simulation time or upon completion of the SimWorld. The Lock Tool prevents the following:

--> removal of ANY SimWorld Objects

--> position change/selection of ANY SimWorld Objects

--> selection of ANY SimWorld Objects

--> addition of ANY Wall and Ground/Intersection Objects

--> colour change of ANY Ground/Intersection Objects

How to Use:

--> To ENABLE the Lock Tool, click in the check box. (the image below indicates an ENABLED Lock Tool)

--> To DISABLE the Lock Tool, click the check box to remove the check. (the image at the top of the page indicates a DISABLED Lock Tool)
