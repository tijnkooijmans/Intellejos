Uploading Controller File

Start by opening a Controller File in a Code Editor window:

1.) To open click: Tools->Code Editor->File->Open

NOTE: ALL of the Controller Files should be stored in the controllers directory

If any changes are made to the Controller File after it is opened it should be saved first.

2.) To save the Controller File: File->Save or File->Save As

In order for the Controller File to upload properly onto the Real Robot it must be compiled with the LEJO COMPILER:

3.) To compile with Lejos: leJOS->Compile

If the popup external processes message window is empty that means the Controller File compiled properly. If there is writing in the popup external processes message window, then there are errors in the Controller File and it will not upload properly. Please correct the errors before uploading.

Once the Controller File compiles properly, turn the Real Robot ON by pressing the On-Off button, place it near the IR TOWER and start the upload by:

4.) To upload the Controller File: leJOS->Link & Upload

A successful upload will yeild ALL of the following:

--> an external process messages window listing the uploaded classes

--> an external process messages window showing percentages up to 100%

--> the Real Robot will beep once
