Shortcuts/Hotkeys

Bellow is a table outlining all of the shortcuts in Intellego 2.0:

 

LOCATION: FUNCTION: SHORTCUT:
     
Intellego 2.0 -> Open Code Editor Window Alt+C
  -> Open Simulator Window Alt+S

Code Editor: -> New file in new Code Editor Ctrl+N
  -> Open file Ctrl+O
  -> Save file Ctrl+S
  -> Exit Editor Ctrl+E
  -> leJOS Complie F5
  -> Link & Upload F6
  -> Upload leJOS Firmware F7
  -> Java Complie F9
     
Simulator: -> Open SimWorld Ctrl+O
  -> Save SimWorld Instance Ctrl+S
  -> Add Robot Ctrl+R
  -> Start Simulation Alt+P
  -> Stop Simulation Alt+S
  -> Undo Ctrl+Z
  -> Redo Ctrl+Y
  -> Grid -> Colour Alt+1
  -> Grid -> Attributes Alt+G
  -> Colour Code -> SimWorld Background Alt+2
  -> Colour Code -> Walls Alt+3
  -> Colour Code -> Road Alt+4
  -> Colour Code -> Robot Alt+5
  -> Colour Code -> Light Sensors Alt+6
  -> Colour Code -> Touch Sensors Alt+7
  -> Light Sensor Properties Ctrl+P
