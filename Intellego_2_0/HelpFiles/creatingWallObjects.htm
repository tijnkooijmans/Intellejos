Creating/Moving Wall Objects

Creating:

Wall Objects can be created by one of two ways (assuming a Simulator window is open, with an active SimWorld):

--> In the Simulator window Click: Options->Build Objects->Wall

- - - OR - - -

--> In the Simulator window Click: The brick ICON on the menu bar, beside the 'Building Tools' label (refer to image below)

After, either of the two creation methods are performed a Wall Object will apper in the SimWorld. The colour of ALL Wall Objects is the SAME, in order to easily identify all objects of this type.

Moving:

1.) Place the cursor over top the Wall Object that is to be moved.

2.) Press the LEFT mouse button DOWN

3.) Drag (combination of holding down the mouse button and moving the mouse) the Wall Object to a desired location on the SimWorld canvas

4.) Release the PRESSED LEFT mouse button when a suitable location is found for the object
