Using the Grid Setup Editor

The grid setup editor may ONLY be accessed once a SimWorld File is opened within a Simulator Window.

--> to open the grid setup editor click: Options->Grid->Attributes

A menu will popup in the 'Property Settings' area, titled 'Grid Setup.' (image below)

--> the 'Grid Spacing' dropdown menu, sets the distance between grid points

--> the 'Pixel Diameter' dropdown menu, sets the diameter of each grid point

After you made the desired setup changes:

--> to apply the grid settings in the SimWorld and close the editor click: ->OK

--> to apply the grid settings in the SimWorld and keep the editor open click: ->Try

--> to abort the last maked grid settings click: ->Cancel

NOTE: Clicking ->Cancel will not undo the changes made by clicking ->Try
