Ground/Intersection Objects

Ground/Intersection Objects provide the simulated robot with road structures. The robot can move freely on top of these objects. The main purpose of Ground/Intersection Objects is to provide functionality to the LightSensors on the robot and allow them to retreive colour readings.

NOTE: Only setting the colors of the Ground/Intersection Objects will not properly set up the SimWorld for LightSensor READINGS. Values must be mapped to colours via the Light Sensor Properties dialog (please follow the link below to learn about proper Light Sensor set up).
