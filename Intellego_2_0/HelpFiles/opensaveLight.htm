Saving/Opening LightSensor Colour Values

Introduction:

The purpose of Saving/Opening colour mappings it to avoid the tedious process of imputing KNOWN values for the colours after colour testing is performed on the Real Robot.

Saving:

1.) map the desired colours to number (click: to learn about mapping colours to numbers)

2.) In the 'LightSensor Properties' dialog click: ->Save

3.) Set the desired file name in the file chooser and click: ->Save

NOTE: ALWAYS save your LightSensor Reading File in the LightSenesorReadings folder

Opening:

1.) In the 'LightSensor Properties' dialog click: ->Open

2.) Select the file in the file chooser and click: ->Open
