Cloning Objects

Wall and Ground/Intersection Objects possess a unique ability to be cloned, meaning you can add an Object to the SimWorld which has the exact same properties as an existing Object.

To clone a Wall or Ground/Intersection Object:

1.) Open the object editor for the Object you want cloned. (to learn about opening the code editor go to: Editing Objects)

2.) Click the button in the top right corner. (image below)

Next, the clone of the selected object will be added to the SimWorld and will appear ON TOP of the selected object.
