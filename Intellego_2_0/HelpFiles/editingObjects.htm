Editing Objects

Wall and Ground/Intersection Objects contain many properties which can be modified. These properties include: exact positioning in the SimWorld, the length, the width and the XZ rotation.

Altering of the above, listed properties is done through a property editor. Not only does the editor allow you to change properties but it allows you to clone your currently selected object (to learn about cloning go to: Cloning Objects). To open this editor:

1.) Place the cursor over top the Wall or Ground/Intersection Oobject that is to be edited.

2.) Click the LEFT mouse button

Next, a menu will popup in the 'Property Settings' area, titled 'Editing:(...)' (image below)

3.) The editable spinners allow setting of property values. The best way to find out what they exactly set is to test them out. Either typing in the value, or clicking on the arrows in the spinners will change their values.

NOTE: the X and Z absolute coordinates of the object are from the CENTER of the edited object and NOT the top left coner.

--> to confirm the changed properties and close the editor click: ->OK

--> to test the changed properties without closing the editor click: ->Try

--> to delete the object from the SimWorld click: ->Del

--> to close the object editor click: ->Cancel
