Rotating the Robot in the SimWorld

To rotate the Robot in the SimWorld:

1.) Left click the SimRCX Object portion (area within the rectangle) of the Robot. (image below)

Next, a selection ring will appear around the robot (diagram provided in part 2.) of Moving the Robot in the SimWorld) and a menu will popup in the 'Property Settings' area, titled 'Robot Bearing:' (image below)

Method 1

2.) Using the spinner select the angle that you would like to set for the robot.

--> to confirm the angle and close the editor click: ->OK

--> to test the angle without closing the editor click: ->Try

--> to close the editor click: ->Cancel

Method 2

2.) Press down the Right mouse button on the SimRCX Object portion of the Robot

3.) Drag the mouse either to the right (it will rotate the robot clockwise) or the the left (it will rotate the robot counter-clockwise).

As you more the drag the mouse horizonally away from the initial click point, the rotation increment will increase proportionally.
