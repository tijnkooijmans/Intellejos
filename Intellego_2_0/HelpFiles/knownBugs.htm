Known Bugs

As the Intellego 2.0 application becomes more widely used I expect this section to grow. Currently the bugs are as follows:

 

ERROR 1: Upon starting the Intellego 2.0 application, the Simulator window will not open and if you refer to the dos window titled 'Intellego_2_0' an error results. (click to see image)

SOLUTION 1: Close the Intellego 2.0 application, and restart it.

 

ERROR 2: (Sometimes) Running multiple simulation sessions in the Simulator window will yeild improper robot behavior.

SOLUTION 2: Run each simulation session in a new Simulator window to ensure proper execution.

 

ERROR 3: During uploading the Real Robot is turned OFF or turns off by itself. The 'external process messages' popup will look like these images.

SOLUTION 3: Turn the Real Robot ON and retry the upload.

 

ERROR 4: During uploading the IR TOWER is not properly connencted. The 'external process messages' popup will look like this image.

SOLUTION 4: Check the connection of the IR Tower to the computer and make sure that it is proper.
