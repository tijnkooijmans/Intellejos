Using the Grid

To assist you in SimWorld creation there is a Grid that can be setup.

NOTE: The grid is designed to help in DESIGNING the SimWorld. As a result the grid should be set to 'NONE' during the execution of the SIMULATION.

The 'How to's' concerning the Grid can be learned by following the links below.
