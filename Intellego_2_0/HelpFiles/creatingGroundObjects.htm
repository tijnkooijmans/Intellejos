Creating/Moving Ground Objects

Creating:

Ground/Intersection Objects can be created by one of two ways (assuming a Simulator window is open, with an active SimWorld):

--> In the Simulator window Click: Options->Build Objects->Ground/Intersection

- - - OR - - -

--> In the Simulator window Click: The green panel ICON on the menu bar, beside the 'Building Tools' label (refer to image below)

After, either of the two creation methods are performed a Ground/Intersection Object will apper in the SimWorld. The colour by default will be either blue or black, but can be edited and each Ground/Intersection Object may have a DISTINCT colour.

Moving:

1.) Place the cursor over top the Ground/Intersection Object that is to be moved.

2.) Press the LEFT mouse button DOWN

3.) Drag (combination of holding down the mouse button and moving the mouse) the Ground/Intersection Object to a desired location on the SimWorld canvas

4.) Release the PRESSED LEFT mouse button when a suitable location is found for the object
