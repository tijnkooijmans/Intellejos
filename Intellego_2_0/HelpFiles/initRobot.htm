Initializing a Robot

To see your robot in the SimWorld, and to run the simulation you must first initialize the robot.

After the Controller File is selected through the file chooser, a menu will popup in the 'Property Settings' area, titled 'Initialize Robot,' as explained in 'Running a Simulation', part '9.)'. (image below)

Four properties of the Robot must be set: the x coordinate, the z coordinate, the initial bearing (xz-clockwise rotation), and the battery voltage of the Robot.

After these values are set via the spinners:

--> to open the robot in the SimWorld click: ->OK

--> to abort the opening click: ->Cancel
