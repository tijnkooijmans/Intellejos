Intellego 2.0 ReadMe

Required Software:

To use the Intellego 2.0 application you must have the software listed below installed:

1.) Intellego 2.0 software

2.) Lejos 2.1 or higher

3.) Java 2 SDK 1.4.0 or higher

 

Intellego 2.0 Setup:

In order to execute the simuation application:

1.) locate the setup MS-Dos Batch file located in the util folder in Intellego 2.0

2.) right click the file and select edit from the menu

3.) set INTELLEGO_HOME= to the absolute path detailing the location of the Intellego 2.0 folder on the comptuer

4.) set LEJOS_HOME= to the absolute path detailing the location of the Lejos folder on the comptuer

5.) set JAVA_HOME= to the absolute path detailing the location of the Java folder on the comptuer

6.) save the updated file and close it

 

Executing Intellego 2.0:

Once the you have the necessary software installed and the software is configured appropriately, to run the simulator simply double-click the Intellego_2_0 MS-Dos Batch file in the main Intellego 2.0 folder.
